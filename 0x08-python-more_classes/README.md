# 0x08. Python - More Classes and Objects
## About
An introductory project on:
- Classes
- Objects
- Instances
- Public, protected and private attributes
- Methods
- __str__ and __repr__
- __dict__
- How to use getattr function
## Requirements
- Python 3.4
- pep8 1.7
